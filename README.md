# WeatherApp.

This is a weather application developed in React that allows you to get updated information about the weather of any city you desire. It uses a weather API to retrieve the necessary data and displays the weather condition with an image and textual description, along with a Fahrenheit temperature indicator.

## Features.

- City Search: You can search for any city in the world using the search bar.
- Updated Information: The app consumes a weather API to fetch the latest and most accurate weather data for the selected city.
- Weather Condition: The app displays an illustrative image representing the current weather condition, giving you a quick      overview of the conditions.
- Textual Description: Along with the image, a textual description of the weather is provided, offering more details about the current conditions.
- Celsius Temperature: In addition to the weather description, the app shows the temperature in Celsius, giving you a clear reference of the current temperature.


## Requirements.

Node.js: Make sure you have Node.js installed on your local machine to run the application.

## Installation.

Follow these steps to run the application locally:

1- Clone this repository to your local machine or download it as a ZIP file and extract it to a folder.

2- Open a terminal and navigate to the root folder of the application.

3- Run the following command to install the necessary dependencies:

```
npm install
```

4- Once the installations are complete, run the following command to start the application:
```
npm start
```

The application will automatically open in your default browser at http://localhost:3000. If it doesn't open automatically, you can manually access that URL.

## Usage.

- On the home page of the application, you'll see a search bar at the top.
- Type the name of the city for which you want to retrieve the weather in the search bar.
- The application will display the current weather information for the selected city, including an illustrative image, a textual description, and the temperature in Celsiius.
- If you want to search for the weather of another city, simply repeat the above steps.
